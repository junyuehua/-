# 清明上河图 · 移动端适配规格

移动端为主流平台，采用「保留结构、简化交互」的思路：砍掉部分功能，但保住寻路与内容。以下为对照桌面端的改动清单。

> 2026-07-13 第一轮已实现（`app/src/MobileViewer.tsx` 等，详见 decisions-log 同日条目）。文中【已定】为实现时拍板补齐的值；【遗留】为本轮未做、待后续的项。

---

## 1. 浏览模式

- 移动端锁死单模式，仅提供默认的第一个模式。
- 不提供模式切换入口。

## 2. 主交互（标识点）

- 桌面端的 hover 改为 **tap**。
- 单步：tap 标识点 → 直接出卡片。不做「预览 → 详情」两步。
- 同一时间只允许一张卡片打开。

## 3. 卡片形态（居中 Modal）

桌面端的旁侧悬停卡片，移动端改为屏幕居中的 modal：

- 居中弹出，背后有半透明遮罩使画面变暗。【已定】遮罩 = `#241607` @ 50%（Figma 219:1681）。
- **关闭方式（并存）**：点遮罩 / Android 系统返回键。关闭后直接回到画面，无其他行为。
- **最大高度**：定死上限，内容超出时 modal 内部滚动，不允许 modal 自身超出屏幕。
- 【已定】**最大宽度**：`min(100vw − 48px, 420px)`——横屏 / iPad / 折叠手机上卡在 420 居中，不摊成宽带。
- **补上下文**：modal 会盖住被点的标识点，用户与画面位置断开。卡片内可放该细节的缩略图，补回「在看什么」。【遗留】缩略图设计稿未画，等定稿再做。

## 4. 底部五段导航栏（原顶部分段栏下移）

- 从顶部移到底部拇指区。
- 功能：五个章节段落的区间跳转 + 高亮当前所在段落。
- 取代原地图导航栏（地图导航栏移动端去掉）。
- ~~**尺寸风险**：五段接近 segmented control 窄屏上限。策略：当前段显示完整章节名，其余段显示序号/缩略。~~【已定 2026-07-13，本条作废】按设计稿（219:1674）五段全名 + **overflow 横向滚动**：栏两端遵守画布 padding 24；内容超宽时通栏滚动，滚到任一尽头仍停在 padding 线上；当前段高亮变化时自动滚回视野。走查时用**最长的章节名**在**最窄机型**上验证，不用占位文字。
- **段间跳转逻辑**：与桌面端保持一致，不做移动端专属设计。

## 5. 寻路与默认缩放

- 寻路完全交给底部五段栏；不再有地图导航栏，也不做连续位置指示。
- **默认缩放级别单独定义**，比桌面更近，保证屏幕上随时可见至少一到两个标识点——让用户靠「下一个点在哪」推进，而非盲拖空白画面。

## 6. 标识点

- 触摸目标最小 44pt（光标级的小尺寸在手指下点不准）。
- 缩放级别下标识点会重叠，需要 clustering / 去重逻辑（桌面端不需要）。
- 【已定 2026-07-13】**tier 全常显**：取消分层显隐门槛（场景 40% / 细节 80% 才浮现），任何缩放下所有点位都在场，密度全交给聚合——小屏默认视野窄，藏点会让用户错过。地标仍恒独立不聚合（导航锚点身份）；场景/细节按屏幕距离聚合。例：同一处 3 细节 + 1 场景 + 1 地标 = 地标独立圈 + 聚合「四」。**同日用户拍板推广到桌面、两端统一，原显隐机制已从代码整体删除**（不再是移动端特例）。
- 【已定】单击/tap 空白画布**无动作**（桌面"单击=放大"不带到移动端），也**不做双击放大**——缩放只靠双指捏合；tap 语义完全留给标识点与聚合。
- 需清晰的**静止态**与**选中态**；没有 hover 态兜底，标识点要在杂乱画面上一眼可读。【遗留】选中态视觉未做（目前"tap 即出 modal"本身是反馈），等设计定稿。

## 7. 布局与控件

- 交互控件放底部（拇指区）；顶部留给不可点的信息（现状：右上角音乐/卷首重开两枚按钮是例外，沿 Figma 219:1674）。
- 画面上的文字加 scrim / 背板（暖调低对比画面 + 户外强光下易糊）。
- 内边距整体收紧。【已定】画布 padding 统一 **24**（卷首页面、modal、底部五段栏共用），叠加 safe-area 取 max。

## 8. 字号与间距

- 重算模块化比例，用更小的基数与更小的比率；**不要**拿桌面字号线性缩放。
- 正文有可读性与 iOS 聚焦缩放的下限约束。
- 【已定】移动字号 token（独立一列，Figma 218:1545/219:1674 为准；差异集中在 `ScrollIntro.module.css` 的 `[data-platform='mobile']` 段）：

  | 元素 | 桌面 | 移动 |
  | --- | --- | --- |
  | 卷首标题（瘦金体） | 64 竖排 | **40 横排**（朱印缩至 34×35 同行） |
  | 卷首正文/引文 | 18 / 行高 2.2 | **16** / 行高 2.2 |
  | 卷首 CTA | 18 / 字距 0.15em | **16**（字距 em 随字号自缩） |
  | 纸签 tab | 14 Bold | 14 不变 |
  | 信息卡标题 / 正文 | 22 / 14 | 22 / 14 不变 |
  | 参考资料 | 14/12/12 | 不变（12 为下限） |
